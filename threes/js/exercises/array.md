map/filter：返回新数组，用于 “转换” 或 “筛选” 元素。
reduce：灵活的累积工具，可实现求和、分组等复杂逻辑。
find：返回第一个符合条件的元素。
some：判断 “是否存在至少一个” 符合条件的元素（部分满足）。
every：判断 “是否所有” 元素都符合条件（全部满足）。
这些方法均为纯函数（不修改原数组），是处理数组的高效工具。

总结
forEach：用于遍历数组并执行操作（无返回值）。
flat(n)：按指定深度扁平化嵌套数组，Infinity 表示完全扁平化。
[...new Set(arr)]：利用 Set 的唯一性快速实现数组去重（适用于基本类型）。
展开运算符 ...：可展开数组元素，用于合并、复制数组等场景（注意是否需要嵌套元素展开）。
